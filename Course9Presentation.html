Simple Linear Models I

We are now going to see different linear models in which the outcome is going to be the mpg variable. In this slide we are going to use as predictors the following variables:

  • Weight (wt)
  • HorsePower (hp)

Simple Linear Models II

We are going to repeat the previous process but this time with the following predictors:

  • Displacement (disp)
  • Rear axle ratio (drat)

Complex linear Models II

We are now going to build a model using the previous predictors.


Call:
lm(formula = mpg ~ wt + drat + disp + hp, data = mtcars)

Coefficients:
(Intercept)           wt         drat         disp           hp
